# Vue3-boilerplate

Simple project template based on the following technologies:

- Vue3 + Typescript
- Vue-router
- Primevue + Primeflex + Primeicons
- Pinia
- Axios
- Vite

IDE Setup:

- VSCode + Volar + TypeScript Vue Plugin + Prettier

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
